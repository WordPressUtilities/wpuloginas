# WPU Login As

Login as another user

How to install
---

* Put this folder to your wp-content/plugins/ folder.
* Activate the plugin in "Plugins" admin section.

How to use
---

* Go to your user list with an administrator level account.
* Click on _login as "userlambda"_
* Navigate where you want.
* When you want to go back, click on the admin bar on the _Back to user "myadmin"_
